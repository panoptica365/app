How to read this

Each segment represents one tenant. Segment size is that tenant's raw event count for today. Segment color is today's count compared to the tenant's own 7-day rolling average:

Green — at or below ~50% above average (normal)
Yellow — 50% to 200% above average (elevated)
Orange — 200% to 500% above average (spike)
Red — more than 500% above average (anomaly)
Gray — insufficient baseline (<1 day of history)

Baselines build up over 7 days. Tenants with a lot of normal activity can still look green even if they dominate the donut; check the tooltip for the actual percentage deviation.
