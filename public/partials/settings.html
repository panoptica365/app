Entra ID Group-Based Access Control
Enforced. Role-based gating is live across every mutate endpoint. Admin manages templates, settings, lifecycle, and audit log. Operator deploys templates, applies security settings, accepts drift, and clears alerts. Viewer is read-only. Every 403 is logged to the MSP audit log.
Paste the Entra ID group Object ID for each role. Panoptica365 resolves the display name via Microsoft Graph when you move focus off the field. Groups live in your own Entra tenant.
