Assign a tier to each tenant from the tenant's edit dialog. Deactivating a tier hides it from new assignments without touching tenants that already have it; a tier can only be deleted once nothing references it.

Assign a rep to each tenant from the tenant's edit dialog. When a rep leaves, deactivate them — existing assignments stay intact until you reassign those tenants; a rep can only be deleted once nothing references them.
